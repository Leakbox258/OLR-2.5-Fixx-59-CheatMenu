### OLR 2.5 Fixx 59 - Спавнер
This is a Cheat Menu MOD for S.T.A.L.K.E.R Oblivious Lost Remake 2.5.<br>
#### Install
##### Fast Start
Overlap `gamedata` dir in this repo on `gamedata` under your orignal game dir.<br>
##### Use JSGME
More Recommanded way: use `JSGME.exe`, create path `MODS\1：OLR 2.5 Fixx 59 - Спавнер\gamedata` (or whatever name you like), then double click `JSGME.exe`
#### How to Use
To Activate the menu, apply `Esc` + `F1` in game.<br>
#### Warning
Some items/npc on the cheat list (which I mean `spawn_list.ltx`) may cause trouble when try to spawn them, especially items with strange name or without any description.<br>
The MOD has tested on Fixx 59, which has been the lasted version announced by `hi_flyer` for a long time. The Fixx 60 may never come out.
